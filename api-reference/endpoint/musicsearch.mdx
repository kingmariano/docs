### Request Body
<ParamField body="song" type="string" required>
Song you want to search for
</ParamField>

<ParamField body="limit" type="string" default="5">
Search Limit
</ParamField>

<ParamField body="proxy" type="string" >
Proxy
</ParamField>

### Response

<ResponseField name="song_name" type="string">
Name of the song recognized.
</ResponseField>
<ResponseField name="shazam_url" type="string">
Url to the song on shazam
</ResponseField>
<ResponseField name="song_image" type="string">
Song cover image
</ResponseField>
